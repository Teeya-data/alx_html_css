# CSS Advanced Project

## Description
This project focuses on implementing advanced CSS styling based on a Figma design. The goal is to create a pixel-perfect webpage using only HTML and CSS, without any external libraries or frameworks.

## Objectives
- Implement a webpage from a designer file (Figma)
- Apply advanced CSS techniques
- Ensure W3C compliance
- Create a responsive and visually accurate design

## Project Structure
- `index.html` - Main HTML file
- `styles.css` - CSS styling file
- `images/` - Directory for image assets

## Design
The design file is available on Figma. Key design elements include:
- Custom fonts: Source Sans Pro and Spin Cycle OT
- Specific color schemes and layouts
- Responsive design considerations
